# camReport classes

A set of S4 and Reference classes to manage camera-trap data and
information.

## Slots

Slots for camReport objects:

- `title`:

  a sdmdata object

- `subtitle`:

  contains the species data

- `reportObjects`:

  a list of modules

- `statusReportObjects`:

  a list of modules
