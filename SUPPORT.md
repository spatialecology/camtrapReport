# Getting help with camtrapReport

Use the GitHub issue tracker for:

- reproducible bugs
- broken outputs
- installation problems that include an error message

Please include:

- your `camtrapReport` version
- your R version
- your operating system
- a minimal reproducible example
- `sessionInfo()`
- the full error message

For usage questions, workflow advice, or interpretation questions, please use Discussions or the package documentation instead of opening a bug report.
