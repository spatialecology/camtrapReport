# Articles

### Tutorials

Step-by-step guides for using camtrapReport.

- [Package
  Overview](https://spatialecology.github.io/camtrapReport/articles/Package-Overview.md):

  An overview of the camtrapReport workflow, package architecture, core
  functionality, and the data quality checks and ecological reports it
  produces.

- [Ecological
  Report](https://spatialecology.github.io/camtrapReport/articles/ecological-report.md):

  An overview of the Ecological Report, its main components, workflow
  steps, and outputs.

- [Data Status
  Report](https://spatialecology.github.io/camtrapReport/articles/data-status-report.md):

  This vignette describes the Data Status Report and explains the checks
  used to assess the quality, completeness, and consistency of
  camera-trap data.

- [Module
  Management](https://spatialecology.github.io/camtrapReport/articles/modules.md):

  This vignette explains how users can manage modules, customize them,
  remove them from the report, change their order, and register and
  integrate new modules into the ecological report.
